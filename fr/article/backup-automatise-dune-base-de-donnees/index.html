Backup automatisé d’une base de données

2018-06-12 Terminal Mac OS Linux

Il est important de mettre régulièrement en place des sauvegardes afin de réduire le risque de perte de données. Cet article va vous permettre de générer une commande BASH pour réaliser un backup entier de votre serveur SQL.
